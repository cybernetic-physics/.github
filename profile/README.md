# Cybernetic Physics

Physical AI development platform. Run Isaac Lab in the cloud, discover reward functions with Eureka, and train reproducible robot policies — no install, no NVIDIA hardware required.

**What we do:**

- **Managed Isaac Lab compute** -- GPU-accelerated robot simulation streamed to your browser via WebRTC. No driver hell, no conda conflicts, no crashes.
- **Eureka reward discovery** -- LLM-guided reward function search built into the training loop.
- **3D reconstruction** -- Upload photos or video, get simulation-ready USDZ assets.
- **MCP integration** -- Drive Isaac Sim sessions from Claude Code or any MCP-compatible environment.
